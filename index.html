ยินดีต้อนรับ

ยินดีต้อนรับทุก ๆ ท่านที่เข้ามาเยี่ยมชมเว็บไซต์นี้นะครับ ผมสร้างเว็บไซต์นี้ขึ้นมาด้วยจุดประสงค์เพื่อที่จะช่วยเพื่อนของผมคนหนึ่งให้สามารถทำความเข้าใจเกี่ยวกับ HTML CSS และ JavaScript ได้ แต่ถ้ามันได้ช่วยผู้อื่นด้วย ผมก็ยินดีครับ และนั่นก็คือสาเหตุที่ผมสร้างเว็บไซต์นี้ขึ้นมาทั้งหมดด้วย HTML CSS และ JavaScript และเปิดให้ทุกคนสามารถเข้าถึง Source Code ของเว็บไซต์นี้ได้ (ผ่านทาง GitHub public repository ของผมเอง)

การเรียนรู้พื้นฐานของทั้งสามภาษาผ่านเว็บไซต์นี้สามารถทำได้หลายวิธี คุณอาจจะเข้าเว็บไซต์นี้ผ่านเบราเซอร์และดูตัวอย่างการเขียนพร้อมคำอธิบายในแต่ละส่วน หรือจะใช้เครื่องมืออย่าง Developer's Tool (DevTools) เพื่อตรวจดูวิธีการเขียนที่ผมใช้และนำไปเป็นแนวทางการศึกษาต่อก็ได้ครับ

ที่สำคัญเลยคือ Website นี้ถูกสร้างขึ้นมาให้มีความ Responsive คุณสามารถเข้ามายังเว็บไซต์นี้ผ่านอุปกรณ์ใด ๆ ก็ได้ที่ต้องการ แต่ผมอยากจะแนะนำว่าถ้าหากเป็นได้ เว็บไซต์นี้ก็ควรจะถูกเข้าชมผ่านอุปกรณ์ที่มี viewport width มากกว่า 400px ครับ

ตัวอย่างโค้ด

ตัวอย่างหลัก ๆ ในเว็บไซต์นี้จะถูกระบุไว้อย่างชัดเจน และสามารถกดปุ่มด้านขวาบนของกรอบตัวอย่างเพื่อคัดลอกตัวอย่างไปใช้งานได้ แต่หากใครต้องการจะดูตัวอย่างผ่าน Source Code ก็อาจจะพบว่า ตัวอย่างเหล่านี้ไม่ได้เขียนอยู่ในรูปแบบที่เว็บไซต์กำลังแสดงผล นั่นก็เพราะว่า เราไม่สามารถเขียนโค้ดจริง ๆ และให้มันแสดงผลดั่งตัวมันเองได้ (สรุปง่าย ๆ ก็คือ ตัวอย่างที่เห็นใช้งานได้จริง และสามารถคัดลอกไปใช้ได้จริง แต่ถ้าเข้าไปดูที่ Source Code ก็อาจจะเจอหน้าตาแปลก ๆ ต่างจากที่เห็นในเว็บไซต์ ไม่ต้องตกใจ)

พื้นฐานความเข้าใจ

ก่อนที่เราจะเริ่มต้นเรียนรู้เกี่ยวกับ HTML CSS และ JavaScript นั้น เราจำเป็นที่จะต้องเข้าใจหลักการการทำงานของเว็บไซต์อย่างง่ายเสียก่อน โดยหัวข้อพื้นฐานความเข้าใจนี้ ผมได้ทำการอธิบายหลักการการทำงานอย่างง่ายเพื่อให้พวกเราเริ่มต้นศึกษาและต่อยอดได้สะดวกมากยิ่งขึ้น โดยผมจะขอแบ่งเป็น 3 หัวข้อ ได้แก่

1. Internet คืออะไร

อินเทอร์เน็ต (Internet) คือชื่อของเน็ตเวิร์ค (Computer Network) ที่มีขนาดใหญ่ที่สุดในโลก คำถามต่อมาก็คือ เน็ตเวิร์คหรือคอมพิวเตอร์เน็ตเวิร์คคืออะไร? อธิบายง่าย ๆ ดังนี้ครับ เมื่อคอมพิวเตอร์ตั้งแต่สองเครื่องขึ้นไปมีการติดต่อสื่อสารกัน ไม่ว่าจะด้วยจุดประสงค์อะไร การติดต่อสื่อสารนั้นก็คือคอมพิวเตอร์เน็ตเวิร์ค (หรือเรียกสั้น ๆ ว่าเน็ตเวิร์ค) นั่นเอง

อินเทอร์เน็ตคือเน็ตเวิร์คที่เชื่อมคอมพิวเตอร์นับล้านเข้าหากัน ทำให้เราติดต่อสื่อสารกันได้ไม่ว่าเราจะอยู่ที่ไหนบนโลก นี่คือความยิ่งใหญ่ของอินเทอร์เน็ต และอินเทอร์เน็ตก็มีบทบาทมากกับการพัฒนาเว็บไซต์ เพราะโดยส่วนใหญ่แล้ว เว็บไซต์ที่เราเข้าชม หรือเว็บไซต์ที่เราจะสร้างเพื่อให้ผู้อื่นเข้าชม จะต้องอาศัยการเข้าถึงผ่านอินเทอร์เน็ต (มีเว็บไซต์บางกลุ่มที่ให้เข้าถึงผ่านการเชื่อมต่อเน็ตเวิร์คเฉพาะอย่าง เช่น private website ของบริษัทผ่าน private network หรือเว็บไซต์ที่ใช้งานสำหรับการสอบในมหาลัย ที่จะไม่เปิดให้บุคคลภายนอกเข้าถึงได้ผ่านอินเทอร์เน็ต)
